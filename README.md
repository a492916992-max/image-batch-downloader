# Image Batch Downloader

A simple utility for downloading images in bulk for datasets, testing, or personal use.

## Features

- Download images from a list of URLs
- Automatically create output folders
- Useful for small dataset building or experiments

## Usage

Scripts are kept simple and easy to modify.
## Notes

This script is intended for learning and small-scale use.
Make sure you have permission to download and use any images.
